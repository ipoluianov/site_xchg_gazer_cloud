Public Gateway Network

What is it?

A set of nodes

XCHG network is in testing mode.

Now only 2 nodes are available:

  • 54.37.73.160 - 500 Megabit/sec
  • 54.37.73.229 - 500 Megabit/sec

    • What services are offered by the nodes of this network?

    Your service is registered on any node of the network. Address of your service is an RSA public key.

    Clients find the host your service is connected to. They send messages to your service.

    Security

    Address protection

    Address is your RSA public key. Only your service can confirm that this is its own address.

    Data encryption

    All traffic between your service and xchg-node is encrypted by AES-256. Key exchange - RSA-2048.

    Traffic between Client and Service encrypted by AES-256. Key exchange - RSA-2048.

    Cost

    All the nodes are free for use. Limited frame size and request frequency.

    You can support the project by donations.

    Add your xchg-node

    You can support the network if you have server with unlimited traffic. Just install xchg and it will be connected to cloud automatically.
