dApps Transport Tools

Direct communication between peers is often a tricky thing. XCHG is a lightweight tool to establish communication between peers using private or public xchg-routers. It might be called PrP (Peer-Router-Peer) - communication between peers through a layer of routers. UDP Hole Punching is used as an important addition.

Features

  • Peer-to-Peer encryption (AES-256/RSA-2048)
  • Public Network of routers
  • Authentication(P2P) is built into the protocol
  • Ability to create your own network of routers for your services
  • SDK for multiple programming languages
  • UDP Hole Punching to speed up transactions
  • Open Source (MIT license) github.com/ipoluianov/xchg
